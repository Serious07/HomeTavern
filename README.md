# HomeTavern

**Ваш личный ИИ-компаньон для ролевых игр и диалогов**

HomeTavern — это приложение, которое позволяет создавать собственных персонажей и вести с ними увлекательные диалоги с помощью искусственного интеллекта. Погрузитесь в ролевые приключения, где каждый персонаж имеет свой уникальный характер, историю и стиль общения.

## Основные возможности

- **🤖 ИИ-персонажи** — Создавайте и настраивайте собственных персонажей с уникальным характером и историей
- **💬 Живые диалоги** — Ведите естественные разговоры с ИИ, который отвечает в стиле вашего персонажа
- **📚 Умная история** — Автоматическое сжатие длинных диалогов для экономии ресурсов (Smart History Compression)
- **🎮 Локальные модели** — Поддержка локальных LLM моделей через llama.cpp — ваши данные остаются на вашем компьютере
- **🌐 Двусторонний перевод** — Автоматический перевод сообщений между русским и английским языками
- **👤 Профиль героя** — Создайте свой персонаж, который будет участвовать во всех диалогах

## Как это работает

HomeTavern использует современные технологии для создания плавного и отзывчивого приложения:

- **Сервер**: Node.js + Express + SQLite
- **Клиент**: React + TypeScript + Tailwind CSS
- **ИИ-движок**: llama.cpp для локального запуска языковых моделей

## Установка

### Шаг 1. Клонирование репозитория

```bash
git clone <url-репозитория>
cd HomeTavern-V5
```

### Шаг 2. Установка зависимостей

```bash
# Установка зависимостей сервера
cd server
npm install

# Возвращаемся в корень и устанавлием клиентские зависимости
cd ../client
npm install

# Возвращаемся в корень
cd ..
npm install
```

### Шаг 3. Настройка переменных окружения

Скопируйте файл примера конфигурации:

```bash
cd server
copy .env.example .env
```

Откройте файл `.env` и настройте необходимые параметры (порт сервера, настройки базы данных и т.д.).

### Шаг 4. Запуск llama.cpp

Для работы с ИИ необходим локальный сервер llama.cpp:

```bash
# Пример запуска с моделью
./llama-server --model ./models/qwen-3.5.gguf --port 8080
```

### Шаг 5. Запуск приложения

Самый простой способ — использовать скрипт запуска:

```bash
npm run dev
```

Или запустить сервер и клиент отдельно:

```bash
# Запуск сервера
npm run server

# Запуск клиента (в отдельном терминале)
npm run client
```

После запуска откройте браузер и перейдите на `http://localhost:4000`

## Быстрый старт

1. **Зарегистрируйтесь** — Создайте учётную запись в приложении
2. **Настройте профиль героя** — Создайте свой персонаж для диалогов
3. **Создайте персонажа** — Добавьте ИИ-компаньера для общения
4. **Начните диалог** — Отправьте первое сообщение и погрузитесь в ролевую игру!
